# 🎯 The Perfect Guess Game

A simple Python number guessing game where the player tries to guess a randomly generated number in the least number of attempts.

---

## 🎮 Game Description

The computer randomly selects a number within a given range, and the player has to guess it.

After each guess, the program provides hints:

* 📈 "Too high" if the guess is greater than the number
* 📉 "Too low" if the guess is smaller than the number
* ✅ "Correct!" when the guess is right

---

## ⚙️ How to Run

1. Make sure Python is installed
2. Run the following command:

```bash
python main.py
```

---

## 🧠 Features

* Random number generation
* User input handling
* Hint system (high/low)
* Attempt counter
* Simple and interactive CLI
